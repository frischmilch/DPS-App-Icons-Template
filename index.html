How to use these templates

This contains a single photoshop template for creating App Icons with one command and an InDesign File to export Splash Screens in required sizes. Includes Icons and Splash Screens for iPad, iPhone (3.5" and 4"), the App Store and Artwork for Android Devices and Google Play.

Template for Viewer Builder Icons.psd

All square icons for iOS and Android derive from the same smart object called “ICON Artwork”.

Open the smart object and insert your own design. Once you edit the smart object, save and come back all icons in this document will change.

The smart object contains another vector smart object that opens in Illustrator.

Template for Splash Screens.indd

The one InDesign CS6 file consists of 6 pages in different sizes. It's straightforward. Follow Export guidelines in slug area. Only tricky thing is Android — you will figure it out!

Export

Export from Photoshop

For a one-click export choose “Save for Web …” in the File menu and choose “All user slices”.

What you get is a nice set of icons ready to use for production in the Viewer Builder

Precision in Photoshop

It looks like Photoshop cannot scale a smart object exactly to a defined size. You will notice a slight grey border on the smaller icons. If it interrupts with your design, make the icons slightly bigger (0,5 px).

Export from InDesign

In InDesign, hit Cmd+E to export. Follow export instructions in the slug area at the top of each page.

Download

Simply hit the download button on the right.
